## Installation & Usage

```
Plug 'astrlux/karonda' " vimscript w/ VimPlug 
use 'astrlux/karonda' -- lua w/ packer
```
```vim
" Vimscript
set termguicolors
colorscheme karonda
```
```lua
-- Lua
vim.opt.termguicolors = true
vim.cmd 'colorscheme karonda'
```
